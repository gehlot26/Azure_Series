# Day-01 - Cloud Computing Terminologies

## What is Cloud ?
Cloud can be defined as a vast, virtual space where we can store files, run software and access various services over the internet.

## What is Cloud Computing ?
- Cloud Computing is a technology model that involves the delivery of computing services over the internet.
- Instead of owning and maintaining physical servers and infrastructure, users can access and use computing resources, applications and storage provided by either third party service providers or their own organization through the internet.

## What is Public Cloud ?
Public cloud can be defined as a giant, shared computer space on the internet. Its like using apps, storing files or doing tasks on the internet that anyone can access.

## What is Private Cloud ?
Private Cloud can be defined as having your own personal, private computer space. Its like a digital clubhouse where only you and your team have access. 

## What is Hybrid Cloud ?
Hybrid Cloud can be defined as having your private computer space but sometimes you use the shared internet space too.

## What is Virtualization ?
Virtualization is a technology that allows you to create multiple simulated environments or virtual instances on a single physical hardware system.

## What is Virtual Machine ?
Virtual Machines are the simulated environments created by the hypervisor. Each virtual machine includes its own virtualized hardware components such as CPU, memory, storage and network interfaces, allowing it to run as an independent operating system and applications.

## What is API ?
Application Programming Interface (API) is a set of rules and protocols that allows different software applications to communicate with each other. It defines how software components should interact.

## What is a Region ?
Regions in cloud computing refer to geographic locations where cloud providers have data centers. Each region contains multiple data centers. 

## What are Availability Zones ?
AZs are isolated locations within a region that have their own power, cooling and networking. They are designed to provide high availability and fault tolerence.

## What is Scalability ?
Scalability is the ability of a system to handle an increasing amount of work or its potential to be enlarged to accommodate that growth.

## What is Elasticity ?
Elasticity in cloud computing refers to the ability to dynamically scale resources up or down based on demand.

## What is Agility ?
Agility is the capability of quickly and easily adapting to changes. In the context of cloud computing, it involves the rapid deployment of resources and applications.

## Whats is High Availability ?
High Availability ensures that a system or application is operational and accessible for high percentage of time, typically 99.9% or higher.

## What is Disaster Recovery ?
Disaster Recovery involves the planning and processes for restoring and recovering data and systems after a natural or human induced disaster.

## What is Load Balancing ?
Load Balancing is the distribution of network traffic or computing workload across multiple servers to ensure no single server is overwhelmed.
